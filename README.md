**Programación Full Stack – UTU 2026**

Este repositorio contiene los trabajos, ejercicios y proyectos realizados durante el año en la materia Programación Full Stack.
El objetivo del repositorio es mantener organizados todos los materiales y prácticas desarrolladas durante el curso.
La estructura del repositorio está organizada en diferentes carpetas según el tipo de contenido o nivel de dificultad de los ejercicios y temas trabajados en clase.

**Organización del repositorio**

El repositorio contiene 10 carpetas principales:

01-Ejercicios-basicos
Contiene ejercicios iniciales y simples que sirven para practicar los conceptos básicos de programación y desarrollo web.

02-Temas-Explicaciones
Incluye ejemplos y materiales utilizados para explicar los diferentes temas vistos durante el curso.

03-Tareas
Carpeta destinada a las tareas y trabajos que se deben entregar durante el año.

04-Ejercicios-Practica
Contiene ejercicios adicionales para practicar y reforzar los conocimientos aprendidos en clase.

05-Temas-MayorImportancia
Aquí se guardan ejemplos y materiales relacionados con los temas más importantes del curso.

06-Ejercicios-avanzados
Incluye ejercicios más complejos que requieren aplicar varios conceptos de programación.

07-Temas-Avanzados
Contiene ejemplos y materiales relacionados con temas avanzados que se verán durante el año.

08-Proyecto-Final
Carpeta destinada al desarrollo del proyecto final de la materia.

09-Ejercicios-Importantes
Incluye ejercicios destacados o relevantes para el aprendizaje de los contenidos del curso.

10-Temas-Complicados
Contiene ejemplos y materiales de los temas que resultan más complejos y que requieren mayor práctica o explicación.

**Motivo de la organización**

Se eligió esta forma de organización para separar los diferentes tipos de trabajos y materiales del curso.
De esta manera es más fácil encontrar ejercicios, tareas o temas específicos a medida que avanza el año, usando su importancia para organizar los temas. Además, esta estructura permite mantener el repositorio ordenado y facilitar la revisión del contenido a lo largo del curso.
